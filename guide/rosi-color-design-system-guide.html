ROSI Design System

Functional minimalism — a mission control terminal, dense with information, calm in color. Blue-gray surfaces. A single muted sage green accent. Depth through tonal layering, not shadows.

Functional Minimal Calm Dense Dark Only
✦
Dark at all times — no light theme. The system has ONE color language: blue-gray neutrals + sage green accent. Only warning (amber) and error (red) break from green. Every element earns its place.

Base — Surfaces

5 tokens

Blue-gray neutrals with warm undertones. Five tonal layers create depth without decoration. Darkest is the browser body. Each step lighter signals elevation. No shadows needed for basic hierarchy.

surface-deepest — Browser body, main content --surface-deepest · #1A1E21
surface-deep — Sidebar, chat area --surface-deep · #1E2327
surface-raised — Cards, inputs, containers --surface-raised · #262B30
surface-overlay — Browser Layer panel --surface-overlay · #2A3038
surface-hover — Hover on all interactive surfaces --surface-hover · #323940

Accent — Sage Green

4 tokens

A single green accent family. Calm enough for sustained reading, strong enough for action signals. Muted is for subtle backgrounds. Strong is the send button and active indicators. One accent, zero confusion.

SwatchTokenHexRoleWhy
--accent #8AB4A0 Active nav, tab labels, links Muted sage. Calm enough to read.
--accent-strong #6BC49F Send button, status dots, RUNNING Strongest signal. Used sparingly.
--accent-muted #3D5A4D Accent card background tint Subtle highlight. Barely there.
--accent-hover #7ECFAA Hover state on accent Brighter. Invites interaction.

Token Naming Convention

Flat, semantic naming. Category first, then variant, then state. No nesting. Machine-readable and human-scannable.

Pattern:
--{category}-{variant}
Examples:
--surface-deepest
--text-secondary
--accent-strong
--sp-4
--radius-md
--motion-base
--z-browser-layer

Design Rules

Hard constraints from the community. Break them only with explicit justification.

Do

Keep it dark — no light theme. Use green accent for interactive/live elements only. Left-align sidebar text. Use uppercase + letter-spacing for section labels. Create depth by tonal layering, not shadows.

Don't

Add colors beyond green accent without functional reason. Use shadows except for overlaying panels and dropdowns. Center-align body text. Use rounded corners larger than 12px except circles. Decorate without purpose.
