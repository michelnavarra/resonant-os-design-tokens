# Changelog

All notable changes to the ROSI Design System.

## [0.2.0] — 2026-05-28

### Changed (from v0.1)
- **Surfaces:** Forest green → blue-gray neutrals (`#1A1E21` → `#323940`)
- **Accent:** Phosphor Green `#4ADE80` → Muted sage green `#8AB4A0`
- **Removed:** Electric Violet accent — one accent only
- **Removed:** Light theme — dark only
- **Typography:** Poppins/Inter/JetBrains Mono → system-ui throughout
- **Success = accent green** — no longer distinct

### Added
- 60 tokens across 12 categories (up from ~30)
- Chrome tokens (tab bar, active tab, toolbar)
- Typography tokens (9: family, sizes, weights, line-height)
- Spacing tokens (8: 2px → 64px scale)
- Radius tokens (4: 4/8/12/16px)
- Elevation tokens (4: shadow levels)
- Motion tokens (4: durations)
- Z-Index tokens (7: layer stacking)
- Interactive visual design system guide (HTML)
- W3C spec `tokens.json`
- CSS custom properties `tokens.css`
- Reference files from live Browser extension source

## [0.1.0] — 2026-05-27

### Added
- Initial color palette (Sokar dashboard era)
- Phosphor Green accent, Electric Violet secondary
- Forest green backgrounds
- Poppins/Inter/JetBrains Mono typography
- ~30 tokens
